# Installation

> This dotfiles is tested only on MacOS

Run

```bash
chmod +x ./scripts/setup.sh
./scripts/setup.sh
```

This script will:

- [ ] Install Homebrew
- [ ] Install all Homebrew packages
- [ ] Install all fish plugins
- [ ] Setup symlinks for configurations
